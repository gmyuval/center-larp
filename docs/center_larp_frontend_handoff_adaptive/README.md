# Center LARP Front-end Handoff — Adaptive Extension

This package extends the original front-end handoff with:

- night / day / auto theme language
- semantic design tokens for mode switching
- responsive and mobile-specific rules
- mobile page blueprints
- starter CSS for themes and breakpoints
- theme toggle JS snippet
- responsive preview HTML files
