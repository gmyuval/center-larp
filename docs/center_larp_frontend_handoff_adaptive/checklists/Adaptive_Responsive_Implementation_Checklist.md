# Adaptive Theme and Responsive Checklist

## Theme system

- [ ] `data-theme` supports `auto`, `day`, and `night`
- [ ] `auto` follows `prefers-color-scheme`
- [ ] manual theme selection persists
- [ ] theme switcher has visible labels and accessible naming
- [ ] day and night palettes both preserve the same brand language
- [ ] night theme is not pure black / generic dark mode
- [ ] focus styles are visible in both themes
- [ ] status colors are not the only state signal

## Responsive system

- [ ] layouts are mobile-first
- [ ] 360px and 390px widths are tested
- [ ] hero remains readable on small screens
- [ ] CTA appears early on mobile landing page
- [ ] logistics stack cleanly on mobile
- [ ] faction rows/cards adapt to one-column layout
- [ ] forms are one-column on mobile
- [ ] buttons and inputs meet touch-target minimums
- [ ] roster entries remain legible and orderly on mobile
- [ ] sticky mobile CTA does not obstruct content
